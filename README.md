# SDL Tennis

Консольная аркадная игра "Теннис" на SDL2.

## Сборка

Требуются SDL2 и SDL2_ttf. Установите и выполните:

```bash
make
./tennis
```

## Управление

- **W / S** — вверх/вниз (игрок)
- **↑ / ↓** — выбор пункта меню
- **← / →** — изменение количества игроков или уровня сложности в меню
- **Enter** — старт игры или перезапуск после игры
- **Esc** — выход
- **↑ / ↓** — вверх/вниз (второй игрок в режиме 2 Players)

## Структура проекта

- `src/` — исходные `.c` файлы
- `include/` — заголовочные `.h` файлы
- `Makefile` — сборка проекта

## Формат коммитов

В проекте используется формат:

```text
type: краткое описание
```

Допустимые типы: `feat`, `fix`, `docs`, `style`, `refactor`, `test`, `build`, `chore`.

Примеры:

```text

fix: ограничить движение бота экраном

```

## Особенности

- Режимы на 1 или 2 игроков
- Простая ИИ логика с 3 уровнями сложности для режима 1 Player
- Меню, счёт, геймовер
- Фоновая музыка и звуковые эффекты
- Используется `SDL2` и `SDL2_ttf`
