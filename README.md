# SDL Tennis

Консольная аркадная игра "Теннис" на SDL2.

## Сборка

Требуются SDL2 и SDL2_ttf. Установите и выполните:

```bash
make
./tennis
```

## Управление

- **W / S** — вверх/вниз (игрок)
- **↑ / ↓** — выбор пункта меню
- **← / →** — изменение выбранной настройки в меню
- **Enter** — применить выбранный пункт меню, старт игры или возврат в меню после игры
- **R** — перезапуск после игры
- **Esc** — выход
- **↑ / ↓** — вверх/вниз (второй игрок в режиме 2 Players)

## Настройки

Настройки хранятся в `config.cfg` рядом с исполняемым файлом:

- `players` — режим `1` или `2` игрока
- `difficulty` — сложность бота от `0` до `2`
- `sound` — `on` или `off`
- `target_score` — счёт до победы: `5`, `10` или `15`
- `player_up`, `player_down`, `second_player_up`, `second_player_down` — клавиши управления

## Структура проекта

- `src/` — исходные `.c` файлы
- `include/` — заголовочные `.h` файлы
- `Makefile` — сборка проекта


Допустимые типы: `feat`, `fix`, `docs`, `style`, `refactor`, `test`, `build`, `chore`.

Примеры:

```text

```

## Особенности

- Режимы на 1 или 2 игроков
- Простая ИИ логика с 3 уровнями сложности для режима 1 Player
- Меню, счёт, геймовер
- Фоновая музыка и звуковые эффекты
- Используется `SDL2` и `SDL2_ttf`
